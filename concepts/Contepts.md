# Oracle Concepts


- CDB, 多租户容器数据库
- DCA, 数据库配置助手
- GDK, Globalization Development Kit, 全球化开发工具集, 10g
- IOT, Index Organized Table, 索引组织表
- OCI, Oracle Call Interface
- 回滚段, 用来提供Oracle数据库过去某个时间点的数据镜像的
