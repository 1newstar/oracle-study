# Oracle Database 18c

## New Feature

- 自治数据库 - Autonomous Database
  - Self-Driving
  - Self-Securing
  - Self-Repairing
- 云级别可用性增强 - Cloud-Scale Availability
- In-Memory内存选件增强
- In-Memory的外部表和InLine外部表支持
- 近似查询 - Approximate Query 和 Top-N 近似聚合
- 机器学习算法新特性
- 多态表支持
- 多租户增强
- 基于NVRAM的多级缓存增强
- 安全增强
